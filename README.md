# coas

Command line tool to see OAS3 file.

## Usage

```
Usage:
  coas [flags]

Flags:
  -b, --build           if specified, builds url interactively
  -c, --config string   config file (default: $HOME/.coas/config.yaml)
  -f, --file string     OAS3 file path
  -h, --help            help for coas
```
